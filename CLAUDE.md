# Project Rules

All development rules are defined in `.claude/rules/`. Each rule is a standalone `.md` file that **must** be read and respected before writing any code.

Before starting any implementation task, read all rule files in `.claude/rules/` and apply them strictly.

## Adding a New Rule

To add a new rule, follow the template defined in `.claude/rules/RULE_TEMPLATE.md` and create a new file in the same folder.
