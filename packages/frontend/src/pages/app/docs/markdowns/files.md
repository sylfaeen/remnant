# File Management

Browse and edit your server files directly from Remnant.

## File Browser

The browser displays the server folder tree:

- **Folders** - Click to open
- **Files** - Click to edit
- **Actions** - Context menu (delete, rename)

## Code Editor

Remnant integrates Monaco Editor (VS Code's editor) for file editing.

### Features

- Automatic syntax highlighting
- Save with `Ctrl+S`
- Unsaved changes indicator
- Support for many formats (YAML, JSON, properties, etc.)
