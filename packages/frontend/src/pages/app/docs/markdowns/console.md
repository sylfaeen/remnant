# Console

The real-time console lets you view logs and send commands to your server.

## Interface

The console page displays:

- **Log area** - Real-time display of server output
- **Command field** - Send Minecraft commands
- **Controls** - Quick Start/Stop/Restart

## Send Commands

Type your command in the bottom field and press `Enter`.

Command examples:

```
say Hello everyone!
op MyUsername
gamemode creative MyUsername
stop
```

::: warning
Commands are executed with server permissions. Be careful with destructive commands.
:::
