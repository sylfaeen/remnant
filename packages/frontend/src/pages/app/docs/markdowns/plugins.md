# Plugins

Manage your Minecraft server plugins.

## Upload Plugins

### Method 1: Drag & Drop

1. Go to the **Plugins** page
2. Drag and drop your `.jar` files into the upload zone
3. Plugins are automatically placed in the `plugins/` folder

### Method 2: Browse Button

1. Click **Browse**
2. Select one or more `.jar` files
3. Confirm the upload

::: warning IMPORTANT
A server restart is required to load new plugins.
:::
