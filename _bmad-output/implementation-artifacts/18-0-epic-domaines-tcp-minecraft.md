# Epic 18: Domaines TCP pour Serveurs Minecraft

L'administrateur peut associer un domaine TCP à un serveur Minecraft. Les domaines TCP sont DNS-only : enregistrements A et SRV pointant directement vers le serveur, sans proxy Nginx.
